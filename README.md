# mb CLI

Command-line interface for MorganaBench.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/morganalabs/mb-cli/main/install.sh | bash
```

To install a specific version:

```bash
MB_VERSION=1.2.3 curl -fsSL https://raw.githubusercontent.com/morganalabs/mb-cli/main/install.sh | bash
```

## Usage

After installation, authenticate with:

```bash
mb login
```

Run `mb --help` for the full command reference.
